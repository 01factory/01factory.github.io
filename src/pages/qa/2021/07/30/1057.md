# Typescript 字符串枚举类型

## 错误的写法

<CodeEditor lan='ts' code={`
function dosomething(type: 'type1'|'type2'){
}\n
const type = 'type1';\n
dosomething(type);
`} />

## 正确的写法

<CodeEditor lan='ts' code={`
function dosomething(type: 'type1'|'type2'){
}\n
const type = 'type1' as const;\n
dosomething(type);
`} />
