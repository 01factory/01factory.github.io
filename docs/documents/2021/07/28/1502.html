开发分类

按开发人员分

这种一般是开发人员自称的分类，大概可以分为

  1. 前端
  2. 后端
  3. 全栈

按应用类型分

  1. web
  2. h5
  3. 各类平台的小程序，如
    1. 微信小程序
    2. 企业微信小程序
    3. 支付宝小程序
    4. 百度小程序
    5. 抖音小程序
    6. QQ小程序
    7. 京东小程序
  4. 快应用
  5. 桌面app
  6. 移动原生app(android/ios)

按开发语言分

编译型语言

  1. C
  2. C++
  3. Delphi
  4. Go
  5. Erlang
  6. Rust
  7. TypeScript
  8. Basic
  9. Swift
  10. Object-C
  11. Dart

解释性(脚本)语言

  1. JavaScript
  2. Python
  3. PHP
  4. Basic

中间语言

即有编译的部分，也有解释部分

  1. Java
  2. C#

  2. 医药
