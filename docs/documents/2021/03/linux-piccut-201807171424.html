  1. 截图软件
    1. 常用软件

使用apt search screenshot查找到Debian软件源中包含的可以在线安装的软件包如下：

xfce4-screenshooter xfce桌面自带的截图工具

imagemagick 图片转换工具

shutter 简单强大的截图工具

gimp linux平台上的PhotoShop

gnome-screenshot 不太好用

screenfetch

screengrab

    1. 安装步骤

sudo apt-get update

sudo apt-get upgrade

sudo apt search screenshot

阅读结果列表，选择期望使用的软件。

使用sudo apt-get install xxx安装。

推荐xfce4-screenshooter+imagemagick/gimp和shutter

其他linux发行版有同样的软件可供选择，软件包可能会有差异。

  1. 使用方法
    1. shutter

打开后有中文界面

打开菜单栏的"编辑--首选项--高级"，修改自动截图前的等待时间，根据情况调整大小。

这个设置主要用在截取的软件或者操作中涉及到下拉菜单，使用以下功能时，可以预留一定的时间去打开子菜单。

待截取的界面示例：

Thunderbird菜单栏的"工具--开发者工具"

shutter菜单栏的"编辑--首选项"

工具栏的其他功能如截取选区/整个屏幕/窗口等不再赘述。

    1. screenshooter

如果使用的是类似windows的cinnamon桌面，可以从菜单搜索xfce4-screen找到xfce4-screenshooter或截图。

如果搜索到两个，执行sudo apt purge gnome-screenshot后再操作。

软件打开后如图

常用的操作有截取整个屏幕/当前窗口/选区，前两种操作可以设置截取前的延迟时间，类似shutter。

  1. 图片处理
    1. ImageMagick

使用 xfce4-screenshooter 截取后会有几种选项，其中"打开方式"一项可以选择工具对截图进行处理。也可以保存后再用ImageMagick打开编辑。

以选择ImageMagick为例，图示操作：

1 选择imagemagick后，在打开的主窗口上点击，左侧或右侧会出现菜单。

2 选择 "Image Edit--Draw",打开编辑界面。

3 选择元素，颜色，线条宽度(粗细)。

分别对应Element，Color，Width，示例给打开方式一行添加了宽度为2的红色方框。

如果需要撤销操作，点击Undo。

如果操作完成，点击Dismiss回到主菜单。

如果要保存，选择"File--save"在弹出窗口通过Up或Home按钮选择保存路径，输入文件名如a1.png。

输入文件名时建议直接输文件名加.png后缀，否则需要使用Format按钮手动选择格式。

    1. gimp

imagemagick用起来不太方便但是轻量。

另一个强大的图片处理工具是gimp。
