  1. 简介

Anaconda 是一个开源的Python发行版本，一种集成较多模块的Python环境。

官网：

https://www.anaconda.com/download/

清华大学镜像站

https://mirrors.tuna.tsinghua.edu.cn/help/anaconda/

  1. 安装
    1. windows

安装文件:

Anaconda3-x.x-Windows-x86_64

FTP /软件仓库/python

双击.exe，下一步下一步，提示需要权限配置环境变量时选择允许，选择设置为默认Python。

如果在安装过程中没有选择好，可以卸载重新安装，或者手动配置环境变量。

  1. 下载anaconda

可以到清华的站点下载相应的版本

  1. 安装

除了安装位置外，还有两个需要确认的地方。

第一个勾是是否把Anaconda加入环境变量，这涉及到能否直接在cmd中使用conda、jupyter、ipython等命令，推荐打勾，如果不打勾话问题也不大，可以在之后使用Anaconda提供的命令行工具进行操作；

第二个是是否设置Anaconda所带的Python 为系统默认的Python版本，这个可选可不选，问题不大。

  1. 验证

安装完成以后，打开cmd测试一下安装结果。

分别输入python、ipython、conda、jupyter notebook等命令，会看到相应的结果，说明安装成功。

（python是进入python交互命令行；ipython是进入ipython交互命令行，很强大；conda是Anaconda的配置命令；jupyter notebook则会启动Web端的ipython notebook）

Anaconda安装成功之后，我们需要修改其包管理镜像为国内源。

  1. 配置镜像地址，否则从官方网站下载升级文件太慢

安装完成后，找到Anaconda prompt，加入镜像地址，完成配置：

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/

conda config --set show_channel_urls yes

    1. Linux
  1. 下载 .sh 后缀的文件。

注意：从FTP下载文件时先设置FTP客户端的文件传输类型为二进制(Binary)

  1. 安装

（参考 [https://docs.anaconda.com/anaconda/install/linux](https://docs.anaconda.com/anaconda/install/linux" \o "https://docs.anaconda.com/anaconda/install/linux)）

bash Anaconda3-4.x-Linux-x86.sh

...

出现 "In order to continue the installation process, please review the license agreement." 时按Enter键。翻页到最底部，输入yes。

出现询问安装路径时，回车选择默认路径(默认～/anaconda3)或CTRL-C 取消安装或手动输入偏爱的路径(路径要存在且当前用户有写入权限)。

出现询问是否将anaconda安装路径加入到环境变量（.bashrc)中，输入yes，默认的是no。或者安装后手动配置环境变量。

出现询问是否安装VSCode时，选择否。

安装完成之后重启终端或 source ~/.bashrc 生效。

或者使用batch模式

bash Anaconda3-5.1.0-Linux-x86_64.sh -b -f -p /opt/anaconda3

  1. 验证

$ python -V

Python 3.x :: Anaconda custom (64-bit)
