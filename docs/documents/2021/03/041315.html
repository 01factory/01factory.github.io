web页面初始化事件

添加初始化事件

使用react钩子函数useEffect，注意第二个参数不能不传，否则会循环不断执行。

()=>{
	const [text, setText] = useState('loading...');
	useEffect(() => {
		setTimeout(()=>{
			setText('loaded successfully!!!');
		}, 3000);
	}, []);
	return <>{text}</>;
}
