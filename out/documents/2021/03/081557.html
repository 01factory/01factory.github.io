Linux 常用命令及操作

1. 文件操作

1.1. pwd 查看当前工作路径

很多命令与当前执行路径有关，所以有时候我们需要知道我们将要执行命令的当前工作路径

pwd

1.2. 查看目录

ls
# or
ls ./01factory

1.3. cd(chdir) 切换到目录

切换到工作目录

cd

切换到上一级目录

cd ..

1.4. md(mkdir)创建目录

在当前工作路径创建名为01factory的文件夹

md 01factory

1.5. mv移动文件或目录/重命名

如果在当前目录下移动文件（夹），相当于重命名。

mv a b

1.6. rm删除文件（夹）

我们在重新安装项目依赖时经常使用它删除全部的依赖库。参数r表示递归删除，即删除所有子目录，f表示强制删除，不提示用户确认。

rm -rf ./node_modules

1.7. 查看文本文件

cat file/path

1.8. 复制文件（夹）

cp file/path new/file/path
# or
cp -r folder/path new/folder/path

2. 常用命令

2.1. 使用管理员权限执行一条命令

可以理解sudo为super user do的缩写

sudo vi

2.2. 软件包安装工具apt

apt指advanced package tool

2.2.1. 查找应用

根据应用名称或部分名称搜索软件包

apt list python
# or
apt list python?
# or
apt list python3*

根据功能搜索软件包

apt search 'web browser'

2.3. 安装软件包

sudo apt install opera -y

2.4. 卸载软件包

sudo apt remove chromium -y

2.5. 更新软件

sudo apt update -y && sudo apt upgrade -y

3. 其它

3.1. 查看本机ip地址

ip a

3.2. 使用vscode打开文件（夹）

code file/path
# or
code path

4. Tilda

Tilda是一个终端模拟器，它能够比较方便地通过某个快捷键（默认为F1）显示和隐藏，不占用桌面窗口数量，且可方便地随时显示/隐藏，是Linux桌面必备的工具之一。

初次使用会自动弹出设置对话框，可以进行一些简单的设置，使得使用起来更得心应手。

  • General

    Start in fullscreen 默认启动全屏显示

    When Last terminal is closed: Open a new terminal and hide 这样就不用当我们关闭终端窗口后多次启动tilda了

  • Appearance

    如果不习惯全屏的使用方式，可以设置以下选项（以百分比方式或者以像素方式）

    Height

    Width
